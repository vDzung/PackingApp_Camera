# Logo Directory

Đặt logo của ứng dụng vào đây.

## Yêu cầu:
- **Tên file:** `logo.png`
- **Format:** PNG (khuyến nghị) hoặc JPG
- **Kích thước:** Tối ưu cho 200x200px (sẽ tự động resize)
- **Nền:** Trong suốt (transparent) hoặc nền trắng

## Cách sử dụng:
1. Đặt file logo vào thư mục này với tên `logo.png`
2. Ứng dụng sẽ tự động load logo khi khởi động
3. Nếu không có logo, hệ thống sẽ hiển thị text logo mặc định
